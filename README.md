## Installation

1. Download the script:
```
wget https://raw.githubusercontent.com/cyberomanov/ironfish-help/main/bms.sh
```
2. Edit script permissions:
```
chmod u+x bms.sh
```
3. Run the script:
```
./bms.sh
```
